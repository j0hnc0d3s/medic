# Medic - Private Clinic Management System

> Built for Capstone Project - UWI Mona Software Engineering 2026

A comprehensive clinic management system for private healthcare providers in Jamaica, featuring patient records, appointment scheduling, financial tracking, and automated reporting.

## 🎯 Project Goals

**Meeting Requirements:**
1. ✅ Store patients in database
2. ✅ Generate weekly reports on patient count
3. ✅ Generate patient record summaries
4. ✅ Input income data
5. ✅ Input expenditure data
6. ✅ Generate financial reports
7. ✅ Send birthday greetings to customers

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Copy environment template
cp .env.example .env

# 3. Add your Firebase credentials to .env

# 4. Start development server
npm run dev
```

Visit http://localhost:5173
